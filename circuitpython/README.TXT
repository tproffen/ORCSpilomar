This folder contains circuitpython code for the Pilomar project.
The tiny2040 subfolder contains the code.py program to run on a Pimoroni Tiny2040 microcontroller. Use CircuitPython not later than 8.2
The tiny2350 subfolder contains the code.py program to run on a Pimoroni Tiny2350 microcontroller. Use CircuitPython 9.2 or later with RP2350 support.

The folder tiny2350-tmc2209 contains code to run the telescope with TMC2209 stepper drivers.
